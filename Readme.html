EasyMounter
Version 0.3

A GUI program for mounting Windows shares.

Actually, a graphical front-end for the command-line program cifsmount.

Note: If you have WON or samba working, you don't need this program.

Features

  • Provides a graphical front-end for the program cifsmount. No messing with the command line! (Unless that's your thing.)

  • Remembers and lists all shares that you create (until you delete them), so you don't have to type in the information over and over again.

Instructions

To add a mount:

  1. Click on Add

  2. Enter the indicated data into each of the text areas:
    • IP: this is the ip address of the remote machine
    • Workgroup: the workgroup of the remote machine
    • Host: the remote machine's name
    • Share: the directory (or printer) that the remote machine is sharing
    • Username: the username you need to access the share (if it doens't need one, just leave it as "user - it cannot be blank)
    • Password: the password you need to access the share (if it doesn't need one, just leave it as "pass - it cannot be blank)
    • Local mount: the local directory you want the share mounted at

  3. Click on Add to add the mount, or Cancel return to the main window.
To edit a mount:
  1. Click on a mount in the list

  2. Click on Edit

  3. Edit the information as neccessary (see To add a mount, above)

  4. Click on Edit to commit the changes, or Cancel to discard them.
